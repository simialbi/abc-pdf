Position Method    

Positions the bottom left of the rectangle.

 

    

Syntax		  

XRect.Position X, Y

 

    

Params		  
Name Type Description
X Double The new left position.
Y Double The new bottom position.

 

    

Notes		  

Moves the rectangle to the supplied position while maintaining the width and height. The corner moved to the location is indicated by the Pin property. The default pin corner is the bottom left.

 

    

Example		  

The following code.

Set rc = Server.CreateObject("ABCpdf12.XRect")
rc.String = "20 20 220 120"
Response.Write "Rect = " & rc
Response.Write "<br>"
rc.Position 50, 50
Response.Write "Pos. = " & rc

Produces the following output.

Rect = 20 20 220 120
Pos. = 50 50 250 150
