XTransform Object    

Represents a world space transform.

When first created, the object defaults to the identity - no transformation.

A world space transform is not an object transform. You are changing the coordinate system - not the objects you're inserting.

 

    
   
Method Description
Invert Inverts the transform.
Magnify Scales about a locked anchor point.
Reset Resets to the identity.
Rotate Rotates about a locked anchor point (angle in degrees).
Skew Skews horizontally and vertically about a locked anchor point.
Translate Translates horizontally and vertically.

 

    
   
Property Default Description
ComFactory No The factory for creating objects in the same application domain.
String Yes A string representation of the transform.
