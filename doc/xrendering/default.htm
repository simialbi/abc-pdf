XRendering Object    

Provides control over PDF rendering and rendering options.

Note: rendering is only available under the ABCpdf Professional License.

Rendering is initiated by using the Save method. Properties of this object allow finer control over the way that rendering is performed.

 

    
   
Method Description
GetData Renders the current area of the current page to memory.
Save Renders and saves the current area of the current page.

 

    
   
Property Default Description
ComFactory No The factory for creating objects in the same application domain.
AntiAliasImages No Whether to anti-alias images.
AntiAliasPoly No Whether to anti-alias polygons.
AntiAliasScene No Whether to apply entire-scene anti-aliasing.
AntiAliasText No Whether to anti-alias text.
AutoRotate No Whether pages should be automatically rotated.
BitsPerChannel No The number of bits per output color channel.
ColorSpace No The name of the output color space.
DefaultHalftone No The default halftone type and options.
DotsPerInch No The output resolution in dots per inch (DPI).
DotsPerInchX No The horizontal output resolution in dots per inch (DPI).
DotsPerInchY No The vertical output resolution in dots per inch (DPI).
DrawAnnotations No Whether to render fields and annotations.
IccCmyk No The path to the default CMYK ICC color profile.
IccGray No The path to the default Gray ICC color profile.
IccRgb No The path to the default RGB ICC color profile.
IccOutput No The path to the default output ICC color profile.
Log No The log for the last render.
NamedSeparation No Named separations.
Overprint No Whether to apply overprint.
SaveAlpha No Whether to save the alpha channel into the output.
SaveAppend No Whether to append to (rather than overwrite) existing image files.
SaveCompression No The preferred compression method.
SaveQuality No The output file quality for lossy compression.
UseEmbeddedHalftone No Whether to use embedded halftones.
