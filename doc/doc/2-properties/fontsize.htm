FontSize Property    

 

Type Default Value Read Only Description
Long   10 No The current text size.

 

    

Notes		  

This property determines the size of text that is added to the document using methods like AddText.

The FontSize property is equivalent to the the XTextStyle.Size property, but unlike the Size property, it only allows integral sizes to be specified.

The font size is measured in the current Units.

 

    

Example		  

The following example adds two blocks of styled text to a document. The first block is in 96-point type, and the second is in 192-point type.

Set theDoc = Server.CreateObject("ABCpdf12.Doc")
theDoc.FontSize = 96
theDoc.AddText "Small "
theDoc.FontSize = 192
theDoc.AddText "Big"
theDoc.Save "c:\mypdfs\docfontsize.pdf"
