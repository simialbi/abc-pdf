AddOval Method    

Adds an oval to the current page.

 

    

Syntax		  

GraphicID = Doc.AddOval(Filled)

 

    

Params		  
Name Type Description
Filled Boolean Whether to fill the oval rather than simply outline it.
GraphicID Long The Object ID of the newly added Graphic object.

 

    

Notes		  

Adds an oval to the current page. The oval is drawn in the current color at the current width and with the current options. It is scaled to fill the current rectangle. The oval may be outlined or filled.

The AddOval method returns the Object ID of the newly added Graphic object.

 

    

Example		  

The following code adds two ovals to a document. The outline oval is semi-transparent.

Set theDoc = Server.CreateObject("ABCpdf12.Doc")
theDoc.Width = 80
theDoc.Rect.Inset 50, 50
theDoc.Color = "255 0 0"
theDoc.AddOval True
theDoc.Color = "0 255 0 a128"
theDoc.AddOval False
theDoc.Save "c:\mypdfs\docaddoval.pdf"
