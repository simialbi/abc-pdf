TransferModule Property    

 

Type Default Value Read Only Description
Long 0 (Default) No The module for obtaining URI/HTML data.

 

    

Notes		  

This property specifies the module to obtain URI/HTML data.

The TransferModule property can take any of the following values:

  • 0 — Default. This is currently the same as RetryNetWebRequestOnEngineAccessDenied.
  • 1 — EngineTransfer. The engine (i.e. MSHTML/WinINet) obtains the page.
  • 2 — NetWebRequest. System.Net.WebRequest is used to obtain the page.
  • 33 — RetryNetWebRequestOnEngineAccessDenied (33 is not 3). The engine obtains the page. If the engine is denied access to local storage, System.Net.WebRequest is used to obtain the page. (Possibly more than one request sent.)
