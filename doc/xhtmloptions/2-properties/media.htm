Media Property    

 

Type Default Value Read Only Description
Long 0 (Print) No The CSS media type to use.

 

    

Notes		  

The Media property can take any of the following values:

  • 0 - Print (default)
  • 1 - Screen

For the Gecko engine, the Screen media must be used with UseScript enabled. An invalid combination (i.e. Screen without UseScript) with yield an exception when you call AddImageUrl or AddImageHtml.

The same HTML document can achieve different visual representation when rendered on different media by specifying several sets of CSS style rules for each medium type. The Gecko engine supports rendering Web pages using the print stylesheets and the screen stylesheets.
