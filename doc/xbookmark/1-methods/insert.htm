Insert Method    

Inserts a new bookmark at a specified location.

 

    

Syntax		  

Set Bookmark = XBookmark.Insert(Index, Title)

 

    

Params		  
Name Type Description
Index Long The zero based index at which the bookmark should be inserted.
Title String The title for the bookmark.
Bookmark XBookmark The bookmark added.

 

    

Notes		  

Use this method to insert a new bookmark at a specified location.

A new bookmark is created and given the title specified. It is added at the specified location, and the newly created bookmark is returned.

If the index is greater than or equal to the number of items in the collection, then the bookmark is appended to the end.

Bookmarks that follow the insertion point move down to accommodate the new bookmark.

 

    

Example		  

None.
