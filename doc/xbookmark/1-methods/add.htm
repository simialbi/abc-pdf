Add Method    

Adds a new bookmark.

 

    

Syntax		  

Set Bookmark = XBookmark.Add(Title)

 

    

Params		  
Name Type Description
Title String The title for the bookmark.
Bookmark XBookmark The bookmark added.

 

    

Notes		  

Use this method to add a new bookmark.

A new bookmark is created and given the title specified. It is added at the end of the collection, and the newly created bookmark is returned.

 

    

Example		  

None.
